Course Introduction:

  • The foundational technologies used in web development i.e. the front-end technologies is the objective of this program.
  • The course elaborates about the 3 core technologies, HTML5, CSS3, and JavaScript.
  • HyperText Markup Language or HTML, is the standard markup language for documents designed to be displayed in a web browser.
  • Cascading Style Sheets or CSS, describes how HTML elements are to be displayed on screen, paper, or in other media i.e. it controls the layout of multiple web pages.
  • JavaScript or JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS.
